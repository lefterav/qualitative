sort_values(mysuite, directory, rep, mode)

source code 

Parallelizes and aligned the vectors provided by the Experiment Suite API

Parameters:
  • mysuite (ExperimentSuite) - instance of the Experiment Suite or subclass
  • directory (str) - the directory that will be searched for results. It needs to have the structure suported by Experiment Suite
Returns:
A dictionary with one entry per experiment. The key of each entry is a tuple containing (attribute_set,classifier_name,discretization,filterscorediff) and its value is a list of (float) values, respectively to their names entered in the list 'required_feature_names'
