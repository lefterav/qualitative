Method Details [hide private]

get_test_sentences(self, filename, filename_train, filename_test, ties_threshold=1, max_sentences=100)

source code 

Splits a judgments set into two sets, one testset and one training set. Allows for extra criteria for the test set, such as no-repetitive judgments and clear ranking (no ties)

Parameters:
  • filename - pointing to the xml file containing the original full set to be read from
  • filename_train - xml file to be created with the training data
  • filename_test - xml file to be created with the test data
  • ties_threshold - test sentences will be dropped, if they have nore than n ties when ranked
