fold_respect_ids(reader, training_writer, test_writer, repetitions, fold, length=None, clean_testset=True)

source code 

An incremental function for cross validation. The function is designed to be executed once for every fold. The current fold is passed as a parameter, so the given xml file is read in a memory efficient way (entry by entry) and then every entry is entered in the training or the testing part accordingly

join_jcml(filenames, output_filename, compact=False)

source code 

Join two XML files, by processing them line-by-line to not overload memory A file id is added in every sentence, to aid with specifying unique ids

Parameters:
  • filenames (list of strings) - a list of filenames that need to be joined
  • output_filename (str) - one filename which will be the result of the joining
  • compact (boolean) - whether the XML should be stripped of the strings
