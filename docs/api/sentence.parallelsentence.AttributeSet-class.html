Class AttributeSet

source code

Known Subclasses:
  • DefaultAttributeSet

Structure that describes the attributes provided by a set of parallel sentences. It is very useful to carry the information for the contents of an entire set, that might otherwise be needed to be parsed incrementally, so this information would not be available.
