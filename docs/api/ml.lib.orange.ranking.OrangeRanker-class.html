Class OrangeRanker

source code

ranking.Ranker --+
                 |
                OrangeRanker

This class represents a ranker implemented over pairwise orange classifiers. This ranker is loaded into the memory from a dump file which contains an already trained model and provides functions to rank one source sentence + translations at a time
